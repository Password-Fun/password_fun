Vanliga frågor

Vi rekommenderar starkt att du inte använder en sida som denna för att skapa lösenord. Försök hitta lösenordshanterare som går att köra på datorn utan internet för att vara extra säker.

Generella regler gällande lösenord

  • Långa lösenord är bättre än komplicerade lösenord. Om du har svårt att komma ihåg längre lösenord så rekommenderar vi att du försöker skapa lösenord från orden i en låt eller något liknande.
  • Vi rekommenderar att du skapar flera lösenord via vår sida även om du bara tänker använda ett lösenord. Om någon skulle kunna se vilka anrop som sker på vår server så är det mycket svårare att gissa om anvädaren skapar flera lösenord och använder endast ett lösenord som skapats.
  • Använd aldrig samma lösenord på olika webbsidor eller enheter.
  • Spara aldrig lösenord i Word dokument eller liknande.
  • Försök att spara alla lösenord lokalt på datorn eller på ett usb minne. Undvik lösningar för att lagra lösenord i molnet.
  • Ändra lösenord ofta
  • Använd aldrig namn på vänner, djur, företag eller liknande när du skapar lösenord. Lösenord som till exempel Företag2016 är enkla att gissa.
  • Fler regler kommer att läggas till i framtiden, skicka gärna förslag till admin@lösenord.xyz eller öppna en issue på Github
